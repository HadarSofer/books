# דף נחיתה לקרמה - בריסטה

## העלאה מהירה ל-Netlify
1. היכנסו ל-netlify.com
2. Add new site → Deploy manually
3. גררו את תיקיית `crema-landing` או את קובץ ה-ZIP
4. תקבלו קישור לאתר תוך כמה שניות

## העלאה ל-GitHub Pages
1. צרו Repository חדש
2. העלו את הקבצים שבתיקייה
3. Settings → Pages → Deploy from branch → main → root

## עריכת טקסטים
הטקסטים נמצאים בקובץ `index.html`.

## החלפת תמונות
התמונות נמצאות בתיקיית `images`.
